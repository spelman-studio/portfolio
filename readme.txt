# Jane Spelman Portfolio Website

## Overview
This repository contains the source code for Jane Spelman’s personal portfolio website. The site showcases design, marketing, and UX/UI projects, with an emphasis on clean front-end development, responsive layouts, and structured project presentation.

---

## Project Goals

- Build a fast, responsive portfolio site using core web technologies  
- Present projects with clear structure and navigation  
- Maintain clean, readable, and scalable front-end code  
- Create a reusable layout for adding future projects  

---

## Tech Stack

- **HTML5** – semantic structure  
- **CSS3** – layout, styling, responsive design  
- **JavaScript (Vanilla)** – interactivity and navigation behavior
